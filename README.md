# 概要
openCVを用いてが画像の特徴点を抽出するPythonプログラムです．各アルゴリズムを用いた際に見つかる特徴点の数も出力できます．
# 使い方
## opencvのインストール
opencvとmatplotlibをpython3にインストールしてください．

```
pip3 install opencv-python
```

## コマンドの実行
### 画像の準備
特徴点を抽出する画像を準備します．

デフォルトでは、現在のディレクトリ直下の`images`というディレクトリに含まれる画像から抽出を行います．

利用できる形式は、`bmp`, `jpg`, `png`です．

### プログラムの実行
プログラムを実行します．
```
python3 fp.py
```
上記コマンドにより、`./images`内の全画像を読み込み，ORBアルゴリズムにより発見された特徴点を描画した画像が`./output`に出力されます．

また、
```
python3 fp.py -c
```
上記のように`-c`オプションを付与することで`countモード`となり，８種類のアルゴリズムで特徴点を抽出した際に発見される特徴点の数を`result.csv`に出力します．

全オプションについては次章で書きます。

## オプション

|オプション名|エイリアス|意味|デフォルト|
|:---:|:---:|:---:|:---:|
|--input_dir|-i|入力ディレクトリの指定|./images/|
|--save_image|-img|出力ディレクトリの指定|./output/|
|--save_log|-log|特徴点数のログをcsvに出力|false|
|--method|-m|抽出アルゴリズムの指定|0|
|--count|-c|countモードに変更|false|

なお，`-log`コマンドは引数として出力されるログファイルの名前を指定できる．

また，`-m`コマンドでは抽出アルゴリズムを番号で指定する．番号と対応するアルゴリズムは以下の通り．

|番号|アルゴリズム|
|:---:|:---:|
|0|ORB|
|1|AGAST|
|2|FAST|
|3|MSER|
|4|AKAZE|
|5|BRISK|
|6|KAZE|
|7|BLOB|
